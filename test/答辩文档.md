# 会议纪要智能体 — 毕业答辩文档

---

## 一、项目概述

### 1.1 项目名称
会议纪要智能体（Meeting Minutes Agent）

### 1.2 项目背景
在日常工作中，会议是企业沟通决策的重要场景。然而，传统的会议纪要整理工作存在以下痛点：
- **人工整理耗时**：一场1小时的会议，人工整理纪要往往需要1-2小时
- **信息遗漏**：口头讨论内容繁杂，关键决策和行动项容易被遗漏
- **格式不统一**：不同人整理的纪要格式各异，难以归档和检索
- **语音转写困难**：会议录音缺乏有效的转写和分析工具

### 1.3 项目目标
本项目旨在构建一个基于 AI 的会议纪要智能处理系统，实现：
- 会议音频的自动语音识别（ASR）
- 会议内容的智能信息提取（NLP）
- 结构化会议纪要的自动生成
- 基于 RAG 的会议内容智能问答
- 支持多格式导出（Word、Markdown）

```python
ACTION_KEYWORDS = [
    "负责", "由.*完成", "安排.*做", "指派", "落实", "执行",
    "推进", "下一步", "上线", "启动", "搞定", "出报告"
]

def extract_decisions(self, text: str) -> list:
    """基于关键词和上下文提取关键决策"""
    decisions = []
    sentences = text.split('。')
    
    for sentence in sentences:
        if any(kw in sentence for kw in DECISION_KEYWORDS):
            # 提取决策内容
            decision = self._extract_decision_content(sentence)
            if decision:
                decisions.append(decision)
    
    return decisions
```

#### 3.4.2 核心实现

```python
class QAService:
    """基于 RAG 的会议智能问答服务"""
    
    def __init__(self):
        # 加载多语言语义编码模型
        self._embedder = SentenceTransformer(
            "paraphrase-multilingual-MiniLM-L12-v2"
        )
    
    def _semantic_search(self, paragraphs: list, question: str, 
                         top_k: int = 3) -> list:
        """语义相似度检索"""
        # 编码问题和段落
        q_embedding = self._embedder.encode(question, convert_to_tensor=True)
        para_embeddings = self._embedder.encode(
            paragraphs, convert_to_tensor=True
        )
        
        # 计算相似度并排序
        hits = util.semantic_search(
            q_embedding, para_embeddings, top_k=top_k
        )[0]
        
        return [paragraphs[hit["corpus_id"]] for hit in hits]
    
    def _generate_answer(self, question: str, context: str) -> str:
        """使用 Qwen3 生成回答"""
        prompt = f"""你是会议纪要助手。根据以下会议文本回答问题。

会议文本：
{context}

问题：{question}

请直接回答："""
        
        # 使用 Qwen3 本地推理
        response = self._call_qwen3(prompt)
        return response
```

### 3.5 音频增强模块 (`audio_processor.py`)

#### 3.5.1 功能描述
对上传的音频进行预处理，提升语音识别准确率。

#### 3.5.2 处理流程

```python
class AudioProcessor:
    """音频增强处理器"""
    
    def enhance_audio(self, input_path: str, output_path: str) -> dict:
        """
        增强音频：降噪 + 音量标准化 + 回声消除
        
        处理流程：
        1. 降噪（noisereduce）
        2. 音量标准化（pydub）
        3. 回声消除（scipy 自适应滤波器）
        """
        # 加载音频
        audio_segment = self._load_audio(input_path)
        
        # 降噪
        if self.has_noisereduce:
            audio_segment = self._denoise(audio_segment)
        
        # 音量标准化
        if self.has_pydub:
            audio_segment = self._normalize_volume(audio_segment)
        
        # 回声消除
        if self.has_scipy:
            audio_segment = self._echo_cancellation(audio_segment)
        
        # 导出
        audio_segment.export(output_path, format="wav")
        return {"success": True, "output_path": output_path}
```

#### 3.5.3 回声消除算法

```python
def _echo_cancellation(self, audio_segment):
    """基于 LMS 自适应滤波器的回声消除"""
    samples = np.array(audio_segment.get_array_of_samples(), dtype=np.float64)
    
    # 参数设置
    delay_ms = 50                    # 预估回声延迟
    delay_samples = int(sr * delay_ms / 1000)
    filter_length = 128              # 滤波器长度
    mu = 0.01                        # 步长参数
    
    # 构造参考信号（延迟版本）
    ref = np.zeros(len(samples), dtype=np.float64)
    ref[delay_samples:] = samples[:-delay_samples]
    
    # LMS 自适应滤波
    weights = np.zeros(filter_length, dtype=np.float64)
    output = np.zeros(len(samples), dtype=np.float64)
    
    for i in range(filter_length, len(samples)):
        x = ref[i - filter_length:i][::-1].copy()
        y = np.dot(weights, x)       # 滤波器输出（回声估计）
        e = samples[i] - y           # 误差信号（消除回声后）
        
        # 更新滤波器权重
        weights += mu * e * x
        output[i] = e
    
    return output
```

### 3.6 说话人识别模块 (`speaker_diarization.py`)

#### 3.6.1 功能描述

#### 3.6.2 技术流程

```
┌─────────────────────────────────────────────────────────┐
│                   说话人识别流程                         │
├─────────────────────────────────────────────────────────┤

│  1. 音频加载与预处理                                     │
│     - 加载音频文件                                       │
│     - 转换为 16kHz 单声道                                │
│                                                         │
│  2. 语音活动检测 (VAD)                                   │
│     - 基于能量阈值的端点检测                              │
│     - 分割为语音段落                                     │
│                                                         │
│  3. 声纹特征提取                                         │
│     - 使用 Resemblyzer 提取说话人嵌入向量                 │
│     - 每个语音段落生成一个 256 维向量                      │
│                                                         │
│  4. 聚类分析                                             │
│     - 使用层次聚类 (AgglomerativeClustering)             │
│     - 自动确定说话人数量                                  │
│     - 输出每个段落的说话人标签                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 3.6.3 核心实现

```python
class SpeakerDiarization:
    """基于声纹特征的说话人识别"""
    
    def diarize(self, audio_path: str) -> list:
        """对音频进行说话人识别"""
        # 1. 加载音频
        audio, duration = self._load_audio(audio_path)
        
        # 2. 语音活动检测
        segments = self._vad_segment(audio, duration)
        
        # 3. 提取声纹嵌入
        embeddings = self._extract_embeddings(audio, segments)
        
        # 4. 聚类分析
        labels = self._cluster_speakers(embeddings)
        
        # 5. 组装结果
        result = []
        for i, (start, end) in enumerate(segments):
            result.append({
                "speaker": f"Speaker_{labels[i] + 1}",
                "start": start,
                "end": end
            })
        
        return result
    
    def _extract_embeddings(self, audio, segments):
        """提取声纹嵌入向量"""
        from resemblyzer import VoiceEncoder
        
        encoder = VoiceEncoder()
        embeddings = []
        
        for start, end in segments:
            # 提取音频片段
            start_sample = int(start * self.sample_rate)
            end_sample = int(end * self.sample_rate)
            segment = audio[start_sample:end_sample]
            
            # 编码为声纹向量
            embedding = encoder.embed_utterance(segment)
            embeddings.append(embedding)
        
        return embeddings
```

```python
class StreamingTranscriber:
    """基于 faster-whisper 的流式语音识别"""
    
    def __init__(self, sample_rate=16000, chunk_duration=3.0):
        self.sample_rate = sample_rate
        self.chunk_duration = chunk_duration
        self.buffer = bytearray()
        self._context_text = ""  # 上下文文本，提升识别连贯性
    
    def add_audio(self, audio_data: bytes):
        """添加音频数据到缓冲区"""
        self.buffer.extend(audio_data)
    
    def should_transcribe(self) -> bool:
        """检查缓冲区是否达到识别阈值"""
        expected_bytes = int(self.sample_rate * 2 * self.chunk_duration)
        return len(self.buffer) >= expected_bytes
    
    def transcribe_chunk(self) -> dict:
        """识别当前缓冲区的音频"""
        # 重采样到 16kHz
        audio_np = self._buffer_to_numpy()
        audio_np = self._resample_to_16k(audio_np)
        
        # 使用 faster-whisper 识别
        segments, _ = self._model.transcribe(
            audio_np,
            language="zh",
            beam_size=5,
            initial_prompt=self._context_text  # 上下文提示
        )
        
        # 拼接结果
        text = "".join([seg.text for seg in segments])
        
        # 更新上下文
        self._context_text = (self._context_text + text)[-200:]
        
        # 清空缓冲区
        self.buffer = bytearray()
        
        return {"text": text, "is_final": False}
```

## 六、关键技术亮点

### 6.1 多引擎语音识别

系统支持两种 ASR 引擎，可根据场景选择：

| 引擎 | 优势 | 劣势 | 适用场景 |
|------|------|------|----------|
| faster-whisper | 免费、本地部署、隐私安全 | 需要 GPU 加速 | 对隐私要求高的场景 |
| MiMo ASR | 识别准确率高、支持多方言 | 需要网络、有调用费用 | 对准确率要求高的场景 |

### 6.2 混合式 NLP 信息提取

采用**规则匹配 + AI 推理**的混合架构：

- **规则匹配**：快速、确定性强、无需调用 API
- **AI 推理**：理解能力强、处理复杂语义

两者互补，既保证了处理速度，又保证了提取准确率。

### 6.3 RAG 智能问答

基于检索增强生成（RAG）的问答系统：

1. **语义检索**：使用 Sentence-BERT 进行向量相似度匹配
2. **上下文组装**：将相关段落作为上下文
3. **生成回答**：使用 Qwen3 本地模型生成自然语言回答

### 6.4 流式处理

- **流式语音识别**：边录边转，实时显示结果
- **流式问答输出**：SSE 协议，逐字输出，提升用户体验

### 6.5 幻觉检测与过滤

针对 Whisper 的幻觉问题，实现了多层过滤：

1. **Prompt Echo 检测**：过滤初始提示词回显
2. **重复检测**：检测并移除重复文本
3. **无意义内容过滤**：过滤过短或无意义的句子
4. **同音字纠错**：修正 ASR 常见错误
