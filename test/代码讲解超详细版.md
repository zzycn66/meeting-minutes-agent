# 第四章：数据库层详解

## 4.1 SQLite 性能优化

```python
@event.listens_for(engine, "connect")
def _set_sqlite_pragma(dbapi_connection, connection_record):
    cursor = dbapi_connection.cursor()
    cursor.execute("PRAGMA journal_mode=WAL")      # WAL 模式
    cursor.execute("PRAGMA synchronous=NORMAL")     # 平衡性能和安全
    cursor.execute("PRAGMA cache_size=-64000")      # 64MB 缓存
    cursor.execute("PRAGMA temp_store=MEMORY")      # 临时表存内存
    cursor.close()
```
**讲解**：

**WAL (Write-Ahead Logging) 模式**：
```
普通模式：
┌─────────┐     ┌─────────┐
│  读操作  │     │  写操作  │
└────┬────┘     └────┬────┘
     └───────┬───────┘
             ▼
        ┌─────────┐
        │  锁冲突  │  ← 读写不能并发
        └─────────┘

WAL 模式：
┌─────────┐     ┌─────────┐
│  读操作  │     │  写操作  │
└────┬────┘     └────┬────┘
     ▼               ▼
┌─────────┐     ┌─────────┐
│  主数据库 │     │  WAL 文件│
└─────────┘     └─────────┘
     └───────┬───────┘
             ▼
        ┌─────────┐
        │  并发OK  │  ← 读写可以并发
        └─────────┘
```

## 5.4 繁简体转换

```python
_T2S_DICT = {
    "會": "会", "來": "来", "對": "对", "開": "开", "關": "关",
    "動": "动", "學": "学", "實": "实", "現": "现", "發": "发",
    "說": "说", "話": "话", "為": "为", "經": "经", "過": "过",
    # ... 约 200 个常用繁体字
}

def _traditional_to_simplified(self, text: str) -> str:
    # 优先使用 zhconv 库
    try:
        from zhconv import convert
        return convert(text, "zh-cn")
    except ImportError:
        pass
    # 兜底：字典映射
    return ''.join(_T2S_DICT.get(char, char) for char in text)
```

## 8.2 语义检索实现

```python
def _semantic_search(self, paragraphs: list, question: str, top_k: int = 3) -> list:
    # 编码段落为向量
    para_embeddings = QAService._embedder.encode(paragraphs, convert_to_tensor=True)
    
    # 编码问题为向量
    q_embedding = QAService._embedder.encode(question, convert_to_tensor=True)
    
    # 计算余弦相似度并排序
    hits = util.semantic_search(q_embedding, para_embeddings, top_k=top_k)[0]
    
    # 返回最相关的段落
    return [paragraphs[hit["corpus_id"]] for hit in hits]
```

**余弦相似度**：
```
余弦相似度 = cos(θ) = (A·B) / (|A|×|B|)

相似度越高（越接近 1），语义越相似
相似度越低（越接近 0），语义越不相关
```

## 9.1 LMS 回声消除算法

```
┌─────────────────────────────────────────────────────────────┐
│                    LMS 自适应滤波器                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  参考信号 x(n) ──┬─────────────────────────────┐           │
│                  │                             │           │
│                  ▼                             │           │
│         ┌───────────────┐                     │           │
│         │  滤波器权重 w  │                     │           │
│         └───────┬───────┘                     │           │
│                 ▼                             │           │
│         ┌───────────────┐                     │           │
│         │  滤波器输出 y  │ (回声估计)          │           │
│         └───────┬───────┘                     │           │
│                 │                             │           │
│                 ▼                             │           │
│  原始信号 d(n) ──┬───────► (+) ────► e(n) ────┤           │
│                  │         ↑                   │           │
│                  │    ─────┘                   │           │
│                  │    ┌───────────────┐        │           │
│                  └───►│  权重更新      │◄───────┘           │
│                       │  w += μ × e × x│                   │
│                       └───────────────┘                     │
│                                                             │
│  输出：e(n) = d(n) - y(n) ≈ s(n)  (消除回声后的信号)       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```python
def _echo_cancellation(self, audio_segment):
    # 参数设置
    delay_ms = 50                      # 预估回声延迟（毫秒）
    delay_samples = int(sr * delay_ms / 1000)
    filter_length = 128                 # 滤波器长度
    mu = 0.01                          # 步长参数（学习率）

    # 构造参考信号（延迟版本，模拟回声）
    ref = np.zeros(len(samples), dtype=np.float64)
    ref[delay_samples:] = samples[:-delay_samples]

    # LMS 自适应滤波
    weights = np.zeros(filter_length, dtype=np.float64)
    output = np.zeros(len(samples), dtype=np.float64)

    for i in range(filter_length, len(samples)):
        x = ref[i - filter_length:i][::-1].copy()  # 参考信号片段
        y = np.dot(weights, x)     # 滤波器输出（回声估计）
        e = samples[i] - y         # 误差信号（消除回声后）
        weights += mu * e * x      # 更新权重
        output[i] = e
```
